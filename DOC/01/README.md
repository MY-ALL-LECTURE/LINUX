#### 플랫폼

|용어|풀이|기타|
|-|-|-|
|PC|범용 시스템|-|
|EMBEDDED|미리 정해진 특정 기능을 수행하기 위한 하드웨어와 소프트웨어가 내장된 시스템|-|
